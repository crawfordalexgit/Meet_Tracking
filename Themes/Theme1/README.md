# CoachesEye Theme Pack (Demo)

This folder contains a minimal Next.js theme demo inspired by the provided dashboard image.

Quick start:

```bash
cd "c:/Users/alex/OneDrive - Mogwai Consultants/00-Tonbridge Swimming/Scripts/Open Meet Dashboard/Themes/Theme1"
npm install
npm run dev
```

Notes:
- Uses Tailwind CSS for utilities. Run `npx tailwindcss init` and ensure PostCSS runs (already configured).
- SVG assets are in `public/` (logo and background). Replace with production assets as needed.
- The `components/` folder contains simple presentational components to replicate the layout.
